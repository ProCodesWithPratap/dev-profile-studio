# Dev Profile Studio

A modern React + Vite + Tailwind CSS app that generates polished GitHub profile README files.

## Features

- Live GitHub profile README markdown generation
- Editable profile fields for name, headline, bio, skills, tools, links, and learning goals
- Built-in profile templates: focused, student, and professional
- Optional GitHub stats, streak card, top languages, and visitor badge sections
- Copy generated markdown to clipboard
- Download generated markdown as `README.md`
- Clean responsive UI built with Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS v4
- Lucide React

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Usage

1. Open the app.
2. Fill in your GitHub profile details.
3. Choose the README sections you want.
4. Copy or download the generated markdown.
5. Paste it into the `README.md` file of your GitHub profile repository.

## Deploy

This project is ready for deployment on Vercel, Netlify, or any static hosting platform that supports Vite builds.
